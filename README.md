# 抢票方法
1.为了方便和简化抢票业务逻辑，提高代码的效率，本抢票软件只负责提交抢票和发起付款请求。<br/>
2.通过手机端将天猫超市端的茅台加入购物车，然后执行下面的操作<br/>
